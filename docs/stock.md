```python
import morningstarFetcher
from pprint import pprint

stock = morningstarFetcher.Stock("0P0001O7D0")  # Initialize Stock with a security ID;

profile = stock.company_profile  # Basic company profile;
print("Company Profile:")
pprint(profile)

insiders = stock.insiders_transaction_history  # Insider transactions history;
print("\nInsiders Transaction History:")
pprint(insiders)  # Display insider transactions history;

```

| Attribute | Description |
